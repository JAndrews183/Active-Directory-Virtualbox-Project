# Active-Directory-Virtualbox-Project
Personal IT project demonstrating Active Directory setup, domain joining, and network configuration using VirtualBox virtual machines.

## Setup Overview
- Windows Server 2022 (Domain Controller: JACORP.local)
- Windows 10 workstation joined to domain
- Static IP configuration & DNS setup
- Active Directory Users, Groups, and Organizational Units
- Group Policy for password complexity & login restrictions

## Tools & Technologies
- Windows Server 2022
- Windows 10
- Oracle VirtualBox
- Active Directory, DNS, Group Policy
